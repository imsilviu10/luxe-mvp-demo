\# Luxe.ro — MVP Demo 18+



Luxe.ro este un MVP demonstrabil pentru o platformă premium 18+ de profiluri, anunțuri, verificare, moderare admin, raportări și plată demo.



Proiectul este construit pentru prezentare, testare și vânzare ca prototip funcțional. Nu este încă o aplicație de producție finală fără backend real, procesator de plăți, upload securizat și verificare legală.
